#DESCRIPTION
This project focuses on developing an interactive dashboard using Tableau for sales data analysis. Multiple visualizations were created to analyze sales, profit, quantity sold, category performance, and regional trends. Different worksheets were combined into a centralized dashboard to provide clear business insights.

The dashboard includes KPI cards, sales trend analysis, category-wise comparison, and geographical sales visualization. Interactive filters and actions were added to improve user experience and enable dynamic analysis based on year, category, and region.

The analysis revealed important insights such as top-performing categories, profitable regions, and yearly sales growth patterns. This dashboard helps users make data-driven decisions efficiently through visual and interactive reporting.
